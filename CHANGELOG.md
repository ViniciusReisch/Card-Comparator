# Changelog

## 0.1.0

- Estrutura open source inicial do projeto.
- Configuracao de Node.js, TypeScript, Vite, React e Express.
- Schema SQLite com cards, offers, monitor_runs, monitor_run_sources e price_history.
- Normalizacao de idioma, estado, preco e chaves canonicas.
- Scrapers Playwright para Liga Pokemon e CardTrader com logs, screenshots e tolerancia a falhas.
- Servicos de monitoramento, diff de ofertas e execucao manual.
- API local para dashboard, cards, ofertas, execucoes e disparo do monitor.
- Interface local com dashboard, novos anuncios, cards, detalhe de card e historico de execucoes.
